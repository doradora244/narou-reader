# 世界観・舞台設定

---

## 世界の基本構造
- **大陸**: 2つ（アルテナ大陸・ザルガ大陸）
- **国家**: 合計10カ国
- **種族**: 人間、獣人、エルフ、ドワーフ、魔族、天翼族（6種族）
- 国同士は常に争っている（領土・資源・種族対立）

---

## アルテナ大陸（6カ国）

| 国名 | 主要種族 | 特色 |
|------|----------|------|
| エルグランド王国 | 人間中心・多種族共存 | 魔法技術が大陸随一 |
| ガルデス帝国 | 人間のみ | 人間至上主義・軍事大国。他種族を格下と見なす |
| ミドーネ共和国 | 人間中心 | 商業・貿易の中心地 |
| セリウス聖王国 | 人間中心 | 宗教国家。神殿勢力が政治を動かす |
| ヴァルタス王国 | 獣人のみ | 人間を敵視。獣人が奴隷（特に性奴隷）として売買されてきた歴史から、人間全般に対し深い憎悪を持つ |
| ルエナ侯国 | 人間・エルフ混在 | 小国。中立外交で生き延びている |

## ザルガ大陸（4カ国）

| 国名 | 主要種族 | 特色 |
|------|----------|------|
| ディアルス | 魔族 | 大陸最大勢力。魔法と武力で支配 |
| エアルム | 天翼族 | 山岳・空域を支配。魔族と全面敵対 |
| シルファリア | エルフ | 森林国家。ドワーフと慢性的に対立 |
| ゴルムヘイム | ドワーフ | 地下都市国家。鍛冶・鉱業の国。エルフと慢性的に対立 |

## 種族間の主な対立関係
- **魔族 ↔ 天翼族**：全面敵対（ザルガ大陸の二大勢力）
- **エルフ ↔ ドワーフ**：資源・領土をめぐる慢性的な対立
- **ガルデス帝国 ↔ エルグランド**：イデオロギー対立（人間至上主義 vs 多種族共存）
- **ヴァルタス王国 ↔ 人間国家全般**：獣人奴隷売買の歴史による深い対立

## 魔物・魔王・悪魔
- **魔物**: 知性を持たない凶暴な生き物。人を襲う。魔族とは別物。
- **魔王**: 伝承のみで実在不明。魔物を使役すると言われている。魔族とは無関係。
- **悪魔**: 世界に稀に「降臨」する存在。伝承では魔王と何らかの関連があるとされているが、実際のところは不明。降臨の条件・目的・正体すべてが謎。

## エルグランド王国
- **規模**: 小国（領土・人口ともに小さい）
- **強み**: 魔法技術が大陸随一
- **特徴**: 他国から魔法を学びに来る者が多く、多種族が共存する国家
- **奴隷制度**: あり。奴隷の多くは他種族。

## 魔法系統
| 系統 | 備考 |
|------|------|
| 炎 | 基本六系統 |
| 水 | 基本六系統 |
| 風 | 基本六系統 |
| 岩 | 基本六系統 |
| 光 | 基本六系統 |
| 闇 | 基本六系統 |
| 特殊魔法 | 六系統に当てはまらない。使い手は基本六系統が使えないことが多い。文献少なく独学困難。貴族社会では金で教師を雇えるため重宝される。 |
| 元素操作 | エクスの適性。前例なし・名前すらない超レア。対象の構成要素を掴んで動かす・混ぜる・分ける・反応させることができる。使い方次第で無限の応用が可能。 |

- 適性は人によって異なる。大体1〜2系統。稀に3〜4、全属性も。
- 多い＝優れているとは限らない。1系統特化や2系統組み合わせも強い。
- **適性検査**: 8歳で教会にて受検。それまで適性は不明。
- **魔法教育**: 一般は8歳から。貴族は専属教師をつける。

## 冒険者ランク
| ランク | 強さの目安 |
|--------|------------|
| G | 入門レベル |
| F | 初心者 |
| E | 一般冒険者 |
| D | 訓練した成人が3〜4年でようやく到達するレベル |
| C | 熟練冒険者 |
| B | パーティ5人がかりでようやく戦えるレベル |
| A | 国家レベルの精鋭と渡り合える |
| S | 一国に一人いるかいないか（ライナ・クロスが史上最年少で達成） |
| SS | 存在自体がおとぎ話に近い |

- 冒険者は基本3〜5人パーティで活動
- エクスは7歳時点でDランク相当（ライナ評価）

## 魔物ランク
- 冒険者ランクと対応している
- Cランクの魔物はCランクパーティ（熟練複数人）を想定した難易度
- Bランクの魔物は熟練冒険者5人パーティが必要。単独対応はSランク以上が必要。

## エルグランド魔法院直属学園
- 12歳入学・六年制（卒業は18歳）
- エルグランド最高峰の魔法教育機関
- 入学できるのはほぼ貴族か相当の実力者。他国からの留学枠もある。
- ルード・アルバスが在籍（入学時から三年目）

## アルバス侯爵家
- エルグランド王国の侯爵家（上から2番目）
- 奴隷の扱いが苛烈なことで有名。他の貴族からも眉をひそめられる。
