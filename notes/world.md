# 世界観・舞台設定

---

## 世界の基本構造
- **大陸**: 2つ（北の大陸・南の大陸）
- **国家**: 合計10カ国
- **種族**: 人間、獣人、エルフ、ドワーフ、魔族、天翼族（6種族）
- 国同士は常に争っている（領土・資源・種族対立）

## 魔物・魔王
- **魔物**: 知性を持たない凶暴な生き物。人を襲う。魔族とは別物。
- **魔王**: 伝承のみで実在不明。魔物を使役すると言われている。魔族とは無関係。

## エルグランド王国
- **規模**: 小国（領土・人口ともに小さい）
- **強み**: 魔法技術が大陸随一
- **特徴**: 他国から魔法を学びに来る者が多く、多種族が共存する国家
- **奴隷制度**: あり。奴隷の多くは他種族。

## 貴族制度（エルグランド）
| 階級 | 称号 |
|------|------|
| 1位 | 公爵 |
| 2位 | 侯爵 |
| 3位 | 伯爵 |
| 4位 | 子爵 |
| 5位 | 男爵 |

- 血筋ではなく魔法技術への貢献で叙爵
- 種族を問わず叙爵可能

## 魔法系統
| 系統 | 備考 |
|------|------|
| 炎 | 基本六系統 |
| 水 | 基本六系統 |
| 風 | 基本六系統 |
| 岩 | 基本六系統 |
| 光 | 基本六系統 |
| 闇 | 基本六系統 |
| 特殊魔法 | 六系統に当てはまらない。使い手は基本六系統が使えないことが多い。文献少なく独学困難。貴族社会では金で教師を雇えるため重宝される。 |
| 元素操作 | エクスの適性。前例なし・名前すらない超レア。対象の構成要素を掴んで動かす・混ぜる・分ける・反応させることができる。使い方次第で無限の応用が可能。 |

- 適性は人によって異なる。大体1〜2系統。稀に3〜4、全属性も。
- 多い＝優れているとは限らない。1系統特化や2系統組み合わせも強い。
- **適性検査**: 8歳で教会にて受検。それまで適性は不明。
- **魔法教育**: 一般は8歳から。貴族は専属教師をつける。

## 冒険者ランク
| ランク | 強さの目安 |
|--------|------------|
| G | 入門レベル |
| F | 初心者 |
| E | 一般冒険者 |
| D | 訓練した成人が3〜4年でようやく到達するレベル |
| C | 熟練冒険者 |
| B | パーティ5人がかりでようやく戦えるレベル |
| A | 国家レベルの精鋭と渡り合える |
| S | 一国に一人いるかいないか（ライナ・クロスが史上最年少で達成） |
| SS | 存在自体がおとぎ話に近い |

- 冒険者は基本3〜5人パーティで活動
- エクスは7歳時点でDランク相当（ライナ評価）

## 魔物ランク
- 冒険者ランクと対応している
- Cランクの魔物はCランクパーティ（熟練複数人）を想定した難易度
- Bランクの魔物は熟練冒険者5人パーティが必要。単独対応はSランク以上が必要。

## エルグランド魔法院直属学園
- 12歳入学・六年制（卒業は18歳）
- エルグランド最高峰の魔法教育機関
- 入学できるのはほぼ貴族か相当の実力者。他国からの留学枠もある。
- ルード・アルバスが在籍（入学時から三年目）

## アルバス侯爵家
- エルグランド王国の侯爵家（上から2番目）
- 奴隷の扱いが苛烈なことで有名。他の貴族からも眉をひそめられる。
